# Icons

Icons are taken from the [Remix Icon](https://remixicon.com/) collection.

## Usage

Icons come in three sizes,

- `large` at `32px`
- `medium` at `20px` and is the default
- `small` at `16px`

1. The `size` prop is available to change the size.
2. The `className` prop is available to change the icon color.
3. All icons except the "File formats" icons will inherit the `currentColor` via CSS. The "File formats" have hardcoded colors that cannot be overridden.

Each icon is a component and is imported from the rollup file `src/icons/index.tsx` that exports all icons.

```jsx
import { IconName } from './src/icons'; // All icon exports

<IconName />; // Icon color will inherit parent `color` value
<IconName size="small" />; // Icon size is small, medium is default
<IconName className="my-class" />; // Icon color can be changed via a class
```

## Adding icons

1. Find the icon you want to use in the [Remix Icon](https://remixicon.com/) collection, click on it, and hit "Copy SVG". Use the "solid" variation for consistency whenever possible.
2. Duplicate a React file in `/src/icons/remix`
3. Rename the file correponsing to the name of the icon
4. Open the new file and rename the component and its export.
5. Paste the SVG code at the bottom of the file and pull out the single long `<path>`. That is the only code you need.
6. Replace the `<path>` in the old SVG with this new one and save.
7. Add a reference to this new icon file in `src/icons/index.tsx`
