# Color

The swatches of most colors are mapped to a palette from light to dark using values from `100` to `900`. This is similar to the way fonts are weighted and is a popular way to organize colors.

Most palettes do not require 9 swatches and values will often be empty. We should only add new ones when we have identified a use case for it in the UserClouds UI.
