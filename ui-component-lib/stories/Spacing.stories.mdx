# Spacing

A system of spacer units is critical to maintain consistent layout spacing both inside and between components. A common approach is to use a `4px` base and increment up by 2x (more or less) at each step. These units are available as SCSS variables and as classes in the Tailwind CSS library.

## Can I use custom spacing?

Occasionally a layout will need a tweak that doesn't align with these units. In high-touch situations you can use custom values.
